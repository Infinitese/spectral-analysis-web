🧮 Step 1 — Coarse Harmonic Residual Scan

We first carry out a grid scan over the range 50 Hz to 400 Hz using 500 equally-spaced candidates. For each trial f₀ we build the U(t,k) matrix with K = 8 harmonics, solve a least-squares amplitude vector a = U\;x, and store the residual energy ‖x − U a‖².

The global minimum of the residual curve provides a robust starting point:

Coarse-scan f0  = 98.20 Hz
Coarse-scan w0  = 617.00 rad/s

(red dot in the residual-versus-frequency plot)

🎯 Step 2 — Adaptive Harmonic LS Refinement

We then refine the estimate with estimate_omega0 (Elvander & Jakobsson, T-SP 2020), which iteratively shrinks the search window until its width drops below 10-12 rad/s. The routine minimises the same least-squares cost but on an adaptive grid.

  • Harmonic order: K = 8
  • Initial grid: 200 points → 50-point local refinements
  • Cost metric: ‖x − U(ω) (U(ω)\x)‖²

Final estimates:

Estimated f0 = 98.205 Hz
Estimated w0 = 617.043 rad/s

These refined values of f₀ / w₀ are used as the baseline for the subsequent OMT-prior inharmonicity analysis.
